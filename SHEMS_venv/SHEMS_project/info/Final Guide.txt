BASIC ENVIRONMENT SETUP
- Python environment creation: 
    folder creation: SHEMS_project as main root and then a single folder for each microservice
    webserver_api folder
        django-admin startproject SHEMS_webserver
        python manage.py startapp mongoDB
        settings.py: 
            INSTALLED_APPS = [..., 'mongoapp'] 
            DATABASES = {
                'default': {
                'ENGINE': 'djongo',
                'NAME': 'SHEMS_test',
                }
            }
        python manage.py makemigrations python manage.py migrate
        certificate and key certificate generation, INSTALLED_APPS = [..., 'mongoapp', 'django_extensions']
            python manage.py makemigrations python manage.py migrate
            python manage.py runserver_plus --cert-file localhost.pem --key-file localhost-key.pem

MONGO DB
How to work with mongoDB:

I'm using 'MongoDB compass' as manager environment, I like it very much. Using it
you need to connect to localhost, just pressing 'connect' without inserting any ip
and creat a new database, only the first time, called 'SHEMS_database'.
then try the code 'mongoDB_client_simulator.py'.
